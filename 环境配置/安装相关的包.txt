打开cmd  输入下列指令

pip install selenium    
（安装浏览器操作包）

pip install Beautifulsoup4
（安装html解析包）

pip install requests
pip install lxml
（xml与html文档解析器）

pip install pytesseract
（pytesseract是google做的ocr库）
（还需要安装tesseract.exe与设置变量，具体看tesseract安装教程）

pip install Pillow
（图像处理库）

pip install PyQt5
（QT图形界面库）
