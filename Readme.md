README
------------------------------------------------------------------------------------
本脚本只是用来学习python过程中设计的，主要用来研究网页操作，实现了用脚本预约某所大学实验平台实验的功能
------------------------------------------------------------------------------------
本程序 需要python3 环境
需要安装几个包       ||       一个验证码识别软件    ||     一个浏览器driver(chromedriver)
具体安装方法见环境配置文件

    默认值设定对着 default_settings.txt 改  
    注意不要加空格和使用中文字符  
    启动方法：当前文件夹下 cmd命令  输入 python Easy_Ui.py
